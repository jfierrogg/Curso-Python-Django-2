Actividad N° 1 – Introducción a la Programación Orientada a Objetos
Módulo: Programación Avanzada en Python

Contenido del proyecto
----------------------

- introduccion_poo.py
  Contiene:
  - Respuestas teóricas en comentarios sobre:
    * Qué es la programación orientada a objetos.
    * Diferencias con la programación estructurada.
    * Ejemplo cotidiano de objeto.
  - Definición de una clase Perro simple.
  - Creación de una instancia y uso del método ladrar().
  - Comentarios que explican:
    * Clase, instancia y objeto.
    * Atributo y estado.
    * Método y comportamiento.
  - Definición de una clase PerroEncapsulado con atributos
    encapsulados (prefijo _) y un método mostrar_info() que
    devuelve el estado del objeto en forma de texto.
  - Comentario sobre abstracción aplicado al ejemplo.

Cómo ejecutar
--------------

1. Abrir una terminal en la carpeta `actividad_m4_l1`.
2. Ejecutar:

   python introduccion_poo.py
3. Se mostrará:

   - El ladrido del primer perro.
   - La información del perro encapsulado.

Reflexión personal
-------------------

Lo que me pareció más interesante de esta actividad fue el cambio de
mentalidad respecto a cómo organizar el código. Antes estaba más
acostumbrado a escribir programas basados en muchas funciones sueltas
y variables globales (un estilo más estructurado). Al trabajar con
clases y objetos, el código queda agrupado de forma más lógica: cada
clase representa una "pieza" del problema y contiene tanto los datos
como las operaciones que le pertenecen.

Esto hace que sea más fácil entender qué hace cada parte, reutilizar
código en nuevos proyectos y extenderlo cuando aparecen nuevos
requerimientos. Además, conceptos como encapsulamiento y abstracción
ayudan a ocultar detalles internos y a exponer solo lo necesario para
usar los objetos, lo que mejora la claridad y el mantenimiento del
programa.
