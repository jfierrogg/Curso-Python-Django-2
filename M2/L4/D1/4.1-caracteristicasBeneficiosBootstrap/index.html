¿Qué es Bootstrap?

Bootstrap es un framework CSS que proporciona estilos y componentes listos para usar, permitiendo crear sitios web responsivos de forma rápida.

Beneficios principales

  • Responsivo por defecto: se adapta a móviles, tablets y escritorio.
  • Componentes reutilizables: botones, navbars, tarjetas, formularios, etc.
  • Ahorro de tiempo: no es necesario escribir todos los estilos desde cero.
